# Zara Sales Analysis

## Project Overview

This project analyzes Zara's product sales data to uncover trends, evaluate product performance, assess promotion effectiveness, and generate business insights. The project combines Python for data preparation, Google BigQuery for SQL analysis, and Power BI for interactive dashboard visualization.

## Business Problem

Zara aims to better understand sales performance across product categories, store sections, promotions, and product placements. The goal is to identify factors influencing sales and provide recommendations to improve business performance.

## Objectives

* Clean and prepare raw sales data using Python.
* Perform business analysis using SQL in Google BigQuery.
* Visualize insights through an interactive Power BI dashboard.
* Generate actionable business recommendations.

## Tools & Technologies

* Python (Pandas, NumPy)
* Google BigQuery (SQL)
* Power BI
* Microsoft Excel
* GitHub

## Dataset

The dataset contains product-level information including:

* Product Name
* Product Category
* Sales Volume
* Promotion Status
* Seasonal Status
* Product Position
* Section (Men/Women)
* Brand Information
* Product Descriptions

## Data Preparation

The following preprocessing steps were performed:

* Removed missing values
* Checked and removed duplicates
* Converted date fields into proper datetime format
* Standardized column names
* Validated data quality
* Exported cleaned dataset for analysis

## SQL Analysis

Key analyses performed include:

* Product Category Analysis

## Key Findings

* Plaid Overshirt was the highest-selling product.
* Promotions showed minimal impact on average sales.
* Products positioned in the aisle generated the highest sales.
* Seasonal and non-seasonal products contributed nearly equally to sales.
* Jackets were the best-performing category.
* The Men's section generated the majority of sales volume.

## Business Recommendations

* Increase inventory for high-performing categories.
* Optimize product placement strategies.
* Reassess promotional campaigns for effectiveness.
* Focus marketing efforts on top-performing product segments.
* Monitor underperforming categories for improvement opportunities.
